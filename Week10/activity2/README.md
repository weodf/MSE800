# Tic-tac-toe Game

## Project Description

This project is a simple command-line Tic-tac-toe game developed in Python.

The game supports two players. Player X and Player O take turns entering their moves. The program checks whether a player wins, whether the game ends in a draw, and whether the input is valid.

## Features

- Command-line interface
- Two-player gameplay
- 3x3 Tic-tac-toe board
- Input validation
- Winner detection
- Draw detection
- PEP8-style readable code
- Pylint support

## How to Run

```bash
python3 main.py
