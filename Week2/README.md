# Advanced Calculator

This project is a calculator program that supports multiple operators and data types, including:

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Modulo (`%`)
- Factorial (`!`)

## Supported Data Types

- Integer
- Float
- Complex Number (example: `2+3j`)

## Notes

- Factorial is only supported for non-negative integers.
- Modulo is only supported for integers and floats.
- Complex numbers are supported for `+`, `-`, `*`, `/`.

## How to Run

```bash
python calculator.py
